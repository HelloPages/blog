Welcome to my guestbook! Feel free to leave a message, share your thoughts, ask questions, or just say hello. I'd love to hear from you and connect with fellow readers and thinkers.

How to leave a comment:

  1. You'll need a GitHub account to comment
  2. Comments are stored as GitHub issues in this repository
  3. Your comment will appear below once posted
  4. Be kind and respectful to others

Note: Comments are powered by utterances, which uses GitHub issues to store and display comments. This ensures your data stays with you and provides a spam-free commenting experience.

Setup Instructions for Site Owner
  1. Replace YOUR_GITHUB_USERNAME/YOUR_REPO_NAME with your actual GitHub repository
  2. Make sure your repository is public
  3. Install the Utterances app on your repository
  4. Enable Issues in your repository settings
  5. The guestbook label will be automatically created when the first comment is posted
